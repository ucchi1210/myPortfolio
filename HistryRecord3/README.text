README

登録したURLから指定時刻に変更を確認して変更があればメールで通知するソフトウェアです。
みなさんお気に入りのサイトを見に行って更新されてなくてがっかりしたことありませんか？
このソフトならがっかりする事はもうありません
メールがくれば必ずどこかが更新しています。
想定するサイトはなろうやカクヨムなどのサイトです。お気に入りの作品のインデックスURLを登録すれば更新されていればメールが来ます。


注意点
登録時のHTMLの文字列と指定時間の文字列を比較する仕組みの為URLは静的なHTMLの場合のみ有効です。
"C:\testfile"のフォルダが必要ですこのフォルダにURLから取得したｈｔｍｌの文字列を記入したｔｘｔファイルが入ります。
eclipseで起動したらhttp://localhost:8080/loginにアクセス新規登録後はあとは流れで


開発環境
IDE eclipse
JDK　17
Spring　boot	version 2.7.2
Spring security version 5.7
